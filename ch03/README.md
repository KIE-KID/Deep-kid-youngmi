# 내용 정리
* 신경망에서는 활설화 함수로 시그모이드 함수와 ReLU 함수 같은 매끄럽게 변화하는 함수를 이용한다.
* 넘파이 다차원 배열을 잘 사용하면 신경망을 효율적으로 구현할 수 있다.
